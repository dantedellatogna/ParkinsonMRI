Parkinson MRI

Por Dante Della Togna - Universidad Tecnológica de Panamá

Sobre la herramienta:

Esta herramienta tiene como objetivo la detección de la enfermedad de Parkinson por medio de imágenes de Resonancia Magnética gracias al uso de Redes Neuronales Convolucionales 3D. En concreto, se hace uso del modelo pre-entrenado RESNET 3D para la clasificación de archivos tridimensionales.

La SALIDA indica el resultado devuelto por la Red Neuronal. Si el resultado es mayor a 0.5, se considera que el paciente tiene Parkinson. En caso contrario, se considera que el paciente NO tiene Parkinson.

Cómo utilizar:

Esta herramienta se encarga de analizar archivos NIFTI (.nii, .nii.gz). Puedes hacer uso de ella con dos simples pasos:

  1. Haz clic en el botón "Seleccionar Archivo".
  2. Utilzia la barra deslizadora para visualizar los diferentes segmentos del archivo de Resonancia Magnética.
  3. Haz clic en "Analizar" para obtener el resultado.
